| Component                 | Requirement                            | Notes                                               |
| ------------------------- | -------------------------------------- | --------------------------------------------------- |
| **Operating System**      | Linux (Ubuntu 24.04 LTS)               | Consistent environment with backend is recommended. |
| **Node.js**               | Node **18.19+**, **20.x**, or **22.x** | Required for building and running Angular UI.       |
| **Process Manager**       | PM2 (recommended)                      | Helps run Angular UI in production.                 |
| **Git**                   | Latest version                         | Used for fetching frontend source.                  |
| **Optional (Production)** | Nginx or Apache as reverse proxy       | Recommended to serve Angular UI efficiently.        |
